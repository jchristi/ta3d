Installation

Windows

Pour installer TA3D sous Windows XP lancez simplement ta3d-0-5-0.exe et décompresser son contenu dans un nouveau dossier (par exemple C:/ta3d/), si vous avez le programme d'installation, il vous suffit de le lancer et de suivre les instructions à l'écran, puis lancez install.bat avec le CD-ROM de TA dans votre lecteur cdrom (Cela copiera totala1.hpi et totala2.hpi dans le dossier de TA3D).

Vous pouvez aussi copier totala1.hpi et totala2.hpi à la main.

Lancez TA3D.exe pour lancer TA3D. (impressionant, hein?)

Note: En cas de plantage essayez de réduire la qualité de l'eau, celle-ci utilise les shaders 2.0 à partir de "normal".

Linux

Comment compiler TA3D soi-même

Pour compiler TA3D sous Linux vous aurez besoin de CMake 2.6 et des librairies Allegro, AllegroGL 0.4.4 (version incluant des correctifs maison, disponible sur notre site web), FMODex 4 et zlib installées avec leurs fichiers de développement ainsi que le paquet source de TA3D.
Il vous faut aussi le compilateur GCC avec le support du C++(g++).

Pour compiler TA3D tapez dans une console:
# cmake -DCMAKE_INSTALL_PREFIX=/usr ./
# make
# cpack -G le_generateur_de_paquet_que_vous_voulez_si_non_DEB_ou_RPM
après vous pouvez installer le paquet si les dépendances sont satisfaites, ou simplement lancer "make install" en root

Pour installer les fichiers ressources lancez install-sh avec le CD-ROM de TA dans votre lecteur cdrom (Cela copiera totala1.hpi et totala2.hpi dans le dossier de ta3d).
Pour lancer TA3D:
# ./ta3d ou cliquez simplement sur l'icône dans le menu de votre bureau

Comment installer un paquet binaire

Des paquets binaires sont maintenant disponibles pour Linux, pour systèmes 32 et 64 bits. Nous avons aussi nos propres dépôts pour Ubuntu et Mandriva (32 et 64 bits) qui contiennent les dépendances nécessaires (Allegro, AllegroGL, et FMODex). Les dépôts Ubuntu sont ici:
# the stable releases
deb http://www.ta3d.org/apt/ stable main
# the development packages
deb http://www.ta3d.org/apt/ testing main
Pour les utilisateurs de Mandriva, il suffit de cliquer ici:
http://www.ta3d.org/urpmi/ta3d.urpmi-media

Pour finir lancer le script install-sh pour installer les ressources requises depuis les cds de Total Annihilation.

NB: si ces paquets ne fonctionnent pas sur votre système (erreurs à propos de quelque chose qui manque ou qui n'est pas compatible) vous pouvez essayer de compiler TA3D vous-même depuis les sources qui ne sont pas incluses dans les paquets binaires.
Pour lancer TA3D:
# ta3d ou cliquez simplement sur l'icône dans le menu de votre bureau
