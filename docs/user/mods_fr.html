Les Mods

Il y a pleins de mods pour TA qui peuvent être chargés par TA3D.
Pour jouer avec un mod:

- copiez les ressources du mod (*.gp3, *.ufo, *.ccx et autres fichiers de ressources) dans ta3d/mods/nom_du_mod/.
- lancez TA3D et sélectionnez le mod dans le menu Options.
- validez et lancez une partie.

Plusieurs mods nécessitent un fichier ta3d.mod pour fonctionner correctement. Après installation il y a déjà les fichiers ta3d.mod pour plusieurs mods de TA.
Par exemple voici ta3d.mod pour TA:Excess :

[MOD]
{
unit_ext = .fbi;
unit_dir = unitGoK;
weapon_dir = weapGoK;
download_dir = downloadGoK;
}

Ces fichiers peuvent être créés manuellement, vous pouvez aussi les demander sur nos forums.

Il peut y avoir autant de mods que vous le voulez dans /mods/, un seul est actif.
