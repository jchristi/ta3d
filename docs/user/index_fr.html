TA3D a pour but de porter TA dans le monde la vraie 3D et est développé par des fans de TA qui aiment encore y jouer, aucune entreprise ne controle le projet. Nous développons TA3D pendant notre temps libre parce que nous aimons TA.


Cela implique que TA3D utilise des ressources du jeu original Total Annihilation:

Différentes façons de lancer TA3D

- pour fonctionner en mode léger vous aurez besoin de:
totala1.hpi pour les ressources et totala2.hpi pour les cartes.
totala2.hpi peut être remplacé par n'importe quel *.ufo,*.hpi,*.gp3, *.ccx contenant des cartes.
(cela permet de jouer avec les unités de TA)

- pour fonctionner en mode normal il vous faut:
totala1.hpi pour les ressources, totala2.hpi pour les cartes, ccdata.ccx, rev31.gp3 de TA: Core Contingency 3.1c.
(pour jouer avec les unités de TA: Core Contingency)
totala2.hpi peut être remplacé par n'importe quel *.ufo,*.hpi,*.gp3, *.ccx contenant des cartes.
Plusieurs pack de cartes peuvent être mis dans le dossier /ta3d/ simultanéments

- pour faire votre propre cuisine il vous faut:
totala1.hpi pour les ressources.
N'hésitez pas à essayer avec différents mods (placez les fichiers du mod dans /ta3d/mods/).
Tous les formats de TA sont supportés.
Note:ne pas mélanger les mods.

La compatibilité de TA3D avec les mods n'a pas été testée à grande échelle pour le moment (ce n'est que la version v.0.5.0), il se peut que le jeu plante ou que des unités de fonctionnent pas correctement, le but principal restant la compatibilité avec Total Annihilation et ses mods.

Certains mods peuvent nécessiter une petite correction avant d'être utilisables par TA3D, cela est dû à certaines modifications faites au moteur de TA. TA3D n'a pas besoin de ces modifications, et les moddeurs pourront adapter leurs mods en quelques minutes.

TA3D supporte les plateformes suivantes:
Windows 32bits
Windows 64bits
Linux 32bits (x86)
Linux 64bits (x86_64)

Site officiel de TA3D: http://ta3d.org/

N'hésitez pas a allez nous rencontrer sur les forums!
